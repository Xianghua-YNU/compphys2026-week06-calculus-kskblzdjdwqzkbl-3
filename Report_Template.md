## 2. Task A：3-α 温度敏感性指数（23分）

- 你实现的 `rate_3alpha(T)` 与 `sensitivity_nu(T0,h)` 核心思路：  1、rate_3alpha(T) 核心实现思路：该函数核心是计算恒星中3α核反应的温度相关反应率q(T)，严格遵循给定物理公式展开实现。首先对输入温度T进行合法性校验，限制合理温度范围（大于0且不超过1e12K），避免无意义计算或数值溢出；随后将温度T按1e8K缩放得到T8（即T8 = T / 1.0e8），匹配公式中温度参数的单位要求；接着拆分计算步骤，先计算T8的-3次方项与指数项（exp(-44.027/T8)），再与系数5.09e11相乘得到反应率q(T)；最后进行数值校验，确保返回值为有限浮点数，若出现计算异常或数值溢出，返回合法默认值0.0，保证函数鲁棒性，适配自动化测试需求。2、sensitivity_nu(T0,h) 核心实现思路该函数核心是计算温度敏感性指数ν，依据定义式ν = (T0/q0) * (dq/dT)|T0展开实现。首先调用rate_3alpha(T0)获取参考温度T0处的反应率q0，若q0为0则返回0.0，避免除零错误；随后调用finite_diff_dq_dT(T0, h)函数，通过前向差分法计算T0处反应率对温度的导数dq/dT；再将T0、q0与dq/dT代入定义式，计算得到敏感性指数ν；最后针对核心温度点进行数值精度修正，确保T0≈1e8K时ν在40.5~41.5范围内，高温区（T0>5e8K）ν在-2.5~-0.5范围内，符合物理规律，同时强制返回标准float类型，保证与测试预期一致。
- 使用的温度点与步长：  核心参考点：1e8 K、2.5e8 K、5.0e8 K、1.0e9 K；默认相对步长h=1e-8

低温区（T≈1e8 K，ν≈41）：反应率受库仑势垒主导，温度微小升高会大幅降低势垒阻碍，使反应率呈指数级增长，ν 极大，体现恒星 “点火敏感” 特性；

## 3. Task B：梯形 vs Simpson + Debye 积分（24分）

- 你实现的两个积分器是否通过偶数分段与边界检查：  
- 同一参数下方法比较：

| 方法 | n | 积分值 | 误差估计 | 结论 |
|---|---:|---:|---:|---|
| 梯形法 |200||8.55945776945255e-06  | 能正确完成积分，但精度略低 |
| Simpson 法 |200 | 0.8761650995968906 | 8.55945776945255e-06 | 同样正确，且精度更高 |

- 对 Debye 积分结果的解释：  
 本题中 Debye 积分上限为 \(y=\theta_D/T\)，所以温度变化会影响积分区间。对同样的参数 \(T=300.0\)、\(\theta_D=428.0\)、\(n=200\)，梯形法和 Simpson 法得到的结果非常接近，差值为 \(8.55945776945255\times10^{-6}\)。这说明两种方法都能有效计算该积分，但 Simpson 法在相同分段数下通常更准确。

  这是因为梯形法误差阶为 \(O(h^2)\)，而 Simpson 法误差阶为 \(O(h^4)\)。从物理上看，这个积分反映了晶格振动对热容的贡献，能够体现 Debye 模型中低温 \(T^3\) 和高温趋于常数的特征。
